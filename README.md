# Cosmic Canvas Portfolio

A space-themed portfolio website built with React, Vite, and Tailwind CSS.
Featuring a dynamic "Cosmic Canvas" design with interactive star cursors, particle backgrounds, and glassmorphism UI.

## Technologies Used

- **Framework**: React + Vite
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **UI Components**: Shadcn UI
- **Animations**: Custom CSS animations & Framer Motion (if applicable)

## Getting Started

1.  Clone the repository:
    ```sh
    git clone <YOUR_REPO_URL>
    ```

2.  Install dependencies:
    ```sh
    npm install
    ```

3.  Run the development server:
    ```sh
    npm run dev
    ```

4.  Build for production:
    ```sh
    npm run build
    ```

## Features

- Interactive Space Background
- Responsive Navigation with "Sanjay N" Logo Animation
- Projects, Skills, and Experience Sections
- Contact Form
